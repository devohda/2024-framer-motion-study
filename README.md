### 보고서 작성 방법

1. 스크립트 실행

```bash
$ npm run report

# or

$ yarn report
```

스크립트 실행 날짜 기준으로 `./reports` 디렉토리에 `yyyy-mm-dd.md` 파일이 생성됩니다.

2. 생성된 파일에 내용 작성
   날짜, 내용, 정리 링크, 참여자를 작성합니다.
